# Ferns and Petals (FNP) Sales Analysis – Excel Dashboard

This project analyzes sales data from Ferns and Petals (FNP) and presents insights using an interactive Excel dashboard.

##  Project Overview
The analysis includes:
- Total Revenue
- Total Orders
- Average Delivery Time
- Revenue by Occasion
- Revenue by Category
- Monthly Revenue Trends
- Top 5 Products by Revenue
- Top 10 Cities by Orders
- Revenue by Order Time

##  Included Files
- **Excel Dashboard** (main analysis)
- **Datasets** used for calculations
- **PDF of assignment tasks**
- **Screenshot of final dashboard**

##  Tools Used
- Microsoft Excel (Pivot Tables, Charts, Slicers)
- Data Cleaning
- Data Transformation
